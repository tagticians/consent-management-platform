How It Works

When a visitor lands on this page the banner checks for a cookie named cmp_consent. If it doesn't exist the banner is shown. Consent choices are persisted in both a cookie (365-day expiry) and localStorage so downstream scripts can read them synchronously.

Cookie format: cmp_consent=functional:1|analytics:0|marketing:0|personalization:0

localStorage key: cmp_consent (JSON object)
